# ⚡ FlashTasks

> Transform ideas into action. Instantly.

FlashTasks is an AI-powered project and task management platform designed to help individuals, teams, and organizations move from planning to execution with less friction and greater efficiency.

Instead of spending time manually organizing thoughts, FlashTasks converts voice notes, messy text, meeting recordings, and conversations into actionable tasks, helping users focus on what matters most: getting work done.

---

## 🚀 Why FlashTasks?

The smallest actions compound into the biggest outcomes.

Yet many people spend more time trying to remember what needs to be done than actually doing it.

Ideas get forgotten.

Meeting notes get buried.

Action items get lost in conversations.

FlashTasks removes that friction by using AI to transform unstructured information into clear, organized, and actionable work.

---

## ✨ Features

### 📝 AI-Powered Task Creation

- Convert messy text into structured tasks
- Turn voice notes into actionable tasks
- Extract tasks from meeting recordings
- Generate subtasks automatically

### 🎯 Smart Productivity

- AI task prioritization
- Focus mode for deep work
- Quick actions and swipe gestures
- Deadline and reminder management

### 👥 Team Collaboration

- Shared workspaces
- Team task management
- Organization management
- Role-based collaboration

### 🤖 AI Automation

Create workflows that continuously perform actions based on your instructions.

Examples:

- Send reminders automatically
- Generate follow-up actions
- Send task summary emails
- Notify team members
- Review pending tasks
- Prioritize work automatically

### 📧 Email Automation

- Send emails
- Create drafts
- Reply to emails
- Search inboxes
- Generate summaries
- Automated follow-ups

## ⚙️ Local Development

### Clone the repository

```bash
git clone https://github.com/abel-otegbola/flashtasks.git
cd flashtasks
````

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
